====== autostart ======

===== About =====
Any .desktop file in this directory will be set to startup after the user logs in.

===== Technical =====
The files in this directory are moved into [[/etc/xdg/autostart/]] on the system. This is the open desktop standard for autostart items. This method SHOULD work on all desktop environments.
